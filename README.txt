Aplicación web creada con Java en servidor.
Debido a la incertidumbre política en la que vivimos actualmente en España, los partidos Ciudadanos y Podemos piden un cambio en la ley electoral para que todos los votos valgan igual en toda España.
Esta aplicación pretende analizar los resultados de todos los años electorales gracias a un lector de csv. Permite seleccionar entre la ley actual y a la que han propuesto cambiar, y también se puede elegir la circunscripción.
Adicionalmente si uno se registra, se le permite subir su propio fichero csv con el resultado de sus sondeos. Esta característica es ideal para analistas.
